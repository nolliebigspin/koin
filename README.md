# Koin

A simple and minimalist currency conversion app built using React Native and Expo.

## Requirements

- [Bun](https://bun.sh)

## Setup

```sh
bun install
```

## Development

```sh
bun start        # start Metro bundler
bun ios          # run on iOS simulator
bun android      # run on Android emulator
```

## Build for device

```sh
bun build:release
```

Builds a release build and installs it directly on a connected iOS device via Xcode.
